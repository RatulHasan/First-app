The controls are basically anchor tags with a class .carousel-control in each of them. Additionally, we need to add a class .left to the left anchor tag and a class .right to the right anchor tag. Each of the anchor tag should have their href attribute targeted to their parent element's ID which is #myCarousel in this case. You should also add a custom attribute named data-slide to specify which slide to load next. This attributes accepts a string value, i.e.: prev and next
